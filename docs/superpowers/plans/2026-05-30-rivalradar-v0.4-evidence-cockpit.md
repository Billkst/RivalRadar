## 1. Positioning (locked)

NOT "a better competitive report generator". RivalRadar is **"the AI analyst that shows its work, attacks its own conclusions, and tells a PM what move to make next."** Every architecture choice below serves one of those three verbs:

- **shows its work** → live execution stream (Manus-style) + EvidencePill claim-level citation + SelfAuditTrace
- **attacks its own conclusions** → QC validates the *decisions themselves* (full-C) + ContradictionPanel
- **tells a PM what move to make** → DecisionBoard, grounded in the user's stated decision context

- **Decision Test**: PT reads 10 min → states 1 concrete action + cites the evidence that changed their mind.

### 12.4 状态覆盖矩阵(Epic 3/4 前必补 — 修复状态覆盖 3/10)
每面板 × {loading 渐进填充 / empty / error / partial / degraded / insufficient_evidence / failed / cancelled} 定义"用户看到什么":
- **运行中**:左栏每面板专属 skeleton,由对应 SSE node-done 解锁(DecisionBoard←done、Comparison←analyze-done、EvidencePill←evidence-list seed、Contradiction←qc-done);解锁前显示进度提示(Comparison:"分析员正在比较 3 个竞品定价…"随 evidenceCountSnapshots 跳数),非通用 spinner。loading 文案 ≠ empty 文案。
- **0 决策**:DecisionBoard 不隐藏 → 解释性空卡("本轮未形成可建议的行动" + 原因 + 引导回证据"采集到 N 条,见对比矩阵→")。
- **degraded**:StatusBar 琥珀/红 banner"证据不足,以下结论置信度下降";决策项视觉弱化 + 每条 caveat(与正常 supported 可视区分)。
- **insufficient_evidence**:决策面不留空 → 有温度"处境卡片"(为什么没结论 + 已有证据诚实总结 + 主操作 放宽竞品范围/重新发现竞品/查看 N 条证据)。继承 DESIGN.md"灰条+虚线边+斜体不留空不臆测"。
- **原文卡三态**:quote 缺失("原文已抓取但未提取到命中引语"+仍给链接)/ source 不可达(禁用"查看原文"标"来源链接不可用"非死链)/ stale(灰角标"采集于 N 天前";**stale 阈值默认 >90 天**,可调)。
- **ExecutionStream failed/cancelled**:失败节点红+可读原因(非 raw exception)+ 其后灰"未执行";cancelled 保留已完成节点 + 未跑灰"已停止" + 顶"已停止·N 秒前";重试环中断箭头呈"中断"。左栏已填充面板保留(诚实展示已得),未解锁"因运行中断未生成"。
